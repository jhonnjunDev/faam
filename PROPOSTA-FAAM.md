## 2. O QUE É O SISTEMA FAAM

O **FAAM** é um sistema web completo para gestão de asilos, clínicas e fundações de saúde, com as seguintes funcionalidades:

### Módulos Inclusos:

| Módulo | Descrição |
|--------|-----------|
| **Dashboard** | Painel com estatísticas e gráficos em tempo real |
| **Cadastro de Pacientes** | Ficha completa com dados pessoais, saúde e histórico |
| **Relatórios Médicos** | 6 tipos: Evolução, Enfermagem, Prescrição, Laudo, Alta, Óbito |
| **Busca e Filtros** | Pesquisa por nome, CPF, quarto, status |
| **Impressão** | Fichas e relatórios formatados para impressão |
| **Gerenciamento de Usuários** | Cadastro de funcionários com perfis de acesso |
| **Sincronização em Nuvem** | Dados acessíveis de qualquer computador ou celular |

### Perfis de Acesso:

| Perfil | Permissões |
|--------|-----------|
| **Administrador** | Acesso total ao sistema |
| **Médico** | Cadastro de pacientes e relatórios |
| **Assistente Social** | Visualização de pacientes e relatórios |

## 3. TECNOLOGIAS UTILIZADAS

- **Frontend:** HTML5, CSS3, JavaScript
- **Banco de Dados:** Supabase (PostgreSQL em nuvem)
- **Hospedagem:** Servidor Pessoal  (jhonata)
- **Segurança:** Autenticação por código, proteção contra brute force, sanitização de dados

## 4. INFRAESTRUTURA NECESSÁRIA

Para o funcionamento adequado do sistema, é necessário um computador dedicado para:

- Acesso ao sistema pelos funcionários

### Especificações Mínimas do Equipamento:

| Componente | Especificação |

| Placa-mãe | Compatível com o processador |

> **Obs:** A doação será apenas do **gabinete com os componentes internos** (processador, memória, armazenamento, placa-mãe e fonte). Monitor, teclado e mouse ficam por conta da fundação.
